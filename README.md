# Machine Learning

## Overview

This repository contains a collection of end‑to‑end machine learning mini‑projects, covering:
- Supervised learning: classification and regression
- Model evaluation and metrics
- Regularization (Lasso, Ridge)
- Dimensionality reduction (PCA)
- Basic clustering

Below is an overview of the notebooks in this repository.

| Project No. | Project                                                                    | Description                                                     |
| ----------- | -------------------------------------------------------------------------- | --------------------------------------------------------------- |
| 1           | [Credit Card Fraud Detection](1_CreditCard_Fraud_Detection.ipynb)          | Credit card transaction fraud detection (classification).       |
| 2           | [Food Delivery Clustering](2_FoodDelivery_Clustering.ipynb)                | Customer/order segmentation for food delivery using clustering. |
| 3           | [Diabetes Detection](3_Diabetes_Detection.ipynb)                           | Diabetes prediction using classification models.                |
| 4           | [Online Purchase Intent (KNN)](4_Online_Purchase_Intent_KNN.ipynb)         | Online purchase intent prediction using KNN.                    |
| 5           | [Advertisement Budget Regression](5_Advertisement_Budget_LR.ipynb)         | Linear regression on advertising budget vs. sales.              |
| 6           | [Bike Sharing OLS Regression](6_Bike_Sharing_OLS_LR.ipynb)                 | OLS linear regression for bike sharing demand.                  |
| 7           | [Health Insurance Prediction](7_Health_Insurance_Prediction.ipynb)         | Health insurance cost or claim prediction.                      |
| 8           | [Lasso & Ridge Regularization](8_Lasso_Ridge_Regularization.ipynb)         | Linear models with Lasso and Ridge regularization.              |
| 9           | [Heart Disease Prediction](9_Heart_Diseases.ipynb)                         | Heart disease prediction using classification models.           |
| 10          | [SVM Classification](10_SVM_Classification.ipynb)                          | Classification using Support Vector Machines.                   |
| 11          | [Appliance Energy Prediction](11_Appliance_Energy_Prediction.ipynb)        | Appliance energy consumption prediction (regression).           |
| 12          | [Breast Cancer Prediction with PCA](12_Breast_Cancer_Prediction_PCA.ipynb) | Breast cancer prediction with dimensionality reduction (PCA).   |
